Proyectos

BoardCtrl

Una Api dedicada a mejorar la organización y administración de diapositivas para poderlas presentar en diferentes dispositivos al mismo tiempo desde un solo computador. Mediante un CRUD con diferentes métodos y clases se logra una buena gestión. Desarrollado con .NET 8 utilizando el framework ASP.NET Core y Entity Framework Core para el mapeo objeto-relacional facilitando la gestión de bases de datos.

Para la base de dato se desarrollo una base de datos relacional en SQL Server Management. Para el front, se utilizo React utilizando el framework de TailwindCSS

RRHHPerformanceEvaluations

Este proyecto es un formulario en el cual se evalua a un empleado para medir su rendimeinto a lo largo del año que paso. Este esta desarrollado con .NET 8, utilizando ASP.NET Core, Entity Framework Core y QuestPdf. Fue pensado para que una vez que se realizara la evaluación, se generara un Pdf y este fuera enviado por correo electrónico a través de el consumo de la API de Masiv, a el Jefe, un representante de RRHH y a el empleado correspondiente.
